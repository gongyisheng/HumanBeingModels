# HumanBeingModels

## Description
This project aims to build some models of networks to gain an overview of how network works and how people work together.
## Prerequisites
#### Front-End  
- React 17.0.2
#### Back-End  
- JDK 11
- SpringBoot 2.5.6
#### Database
- MySql 8.0.25
## Installation
1. Clone the project code
````
git clone https://github.com/gongyisheng/HumanBeingModels.git
````
2. Install dependencies
````
npm install
````
3. Update your database connections
- Rename application-safe.properties to application.properties
- update datasource.url/username/password in application.properties
4. Start local server
````
npm run start:dev
````
